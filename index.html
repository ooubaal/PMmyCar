การซิงก์ข้อมูล (OneDrive Database)

1. ใช้งานบน PC (บันทึกตรงเข้าเครื่องผ่าน OneDrive Folder)

พร้อมใช้

เชื่อมโยงแอปหน้าเว็บเข้ากับไฟล์ในเครื่องคอมพิวเตอร์ของคุณโดยตรง ไฟล์จะบันทึกอัตโนมัติลงในโฟลเดอร์ OneDrive ของคุณ

สถานะ: ไม่ได้เชื่อมต่อไฟล์โดยตรง (ใช้ฐานข้อมูลชั่วคราวในเบราว์เซอร์)

2. ใช้งานบนมือถือ / คลาวด์ (ซิงก์ตรงกับ OneDrive Cloud)

ไม่ได้เข้าสู่ระบบ

ล็อกอินด้วยบัญชี Microsoft เพื่อดึงและเซฟไฟล์ฐานข้อมูลโดยตรงจากคลาวด์ เหมาะกับการใช้งานบนมือถือเพื่อตัดสลับเซฟไฟล์เดียวกัน

วิธีสร้าง Client ID ฟรีใน 3 นาที (คลิก)

สำรองข้อมูลแบบ Manual

หากไม่สะดวกเชื่อมต่อ OneDrive สามารถกดนำเข้าหรือส่งออกไฟล์ข้อมูลเป็น JSON เพื่อนำไปเปิดในเครื่องอื่นได้ทันที

ข้อมูลทดสอบ

สร้างข้อมูลอะไหล่จำลองและประวัติบำรุงรักษาเพื่อทดลองเล่นหน้าต่างใช้งานได้ทันที

คู่มือ 3 ขั้นตอน: การสร้าง Client ID เพื่อต่อเข้ากับ OneDrive ฟรี

ขั้นตอนที่ 1: เข้าสู่ Microsoft Azure Portal

เข้าไปที่ Azure Portal หรือหน้า Microsoft Entra admin center แล้วเข้าสู่ระบบด้วยบัญชี Hotmail / Outlook / Live ส่วนตัวของคุณได้ฟรี

ขั้นตอนที่ 2: ลงทะเบียนแอปพลิเคชัน (App Registration)
  1. ไปที่เมนู **App registrations** แล้วกดปุ่ม **+ New registration**
  2. **Name**: ตั้งชื่อแอปว่า `PM My Car`
  3. **Supported account types**: เลือกหัวข้อที่ 3: **"Accounts in any organizational directory (Any Microsoft Entra ID tenant - Multitenant) and personal Microsoft accounts (e.g. Skype, Xbox)"** *(สำคัญมากเพื่อให้ใช้ล็อกอินกับอีเมลส่วนตัวได้)*
  4. **Redirect URI**: เลือกรูปแบบเป็น **Single-page application (SPA)** และใส่ช่อง URL เป็นที่อยู่เว็บของคุณ เช่น:
    • หากใช้งานผ่านเครื่องจำลอง: `http://localhost:5500` หรือ `http://localhost:8000`
    • หากใช้งานบน GitHub Pages: `https://[ชื่อคุณ].github.io/pm-my-car/`
    • *หรือสามารถเพิ่มได้หลาย URL ภายหลัง*
  5. กดปุ่ม **Register** เพื่อลงทะเบียน
ขั้นตอนที่ 3: คัดลอก Application (client) ID

เมื่อกดลงทะเบียนแล้ว คุณจะเข้าหน้า Overview ให้คัดลอกรหัส **Application (client) ID** (รหัสยาวๆ ที่ประกอบด้วยขีดหลายๆ ตัว) มาวางใส่ในช่องด้านบนของแอปนี้ และกดยอมรับการอ่าน/เขียนไฟล์ (Permissions > Files.ReadWrite) ก็จะซิงก์ OneDrive ได้โดยตรงทันทีครับ

เพิ่มอะไหล่ใหม่ที่ต้องการติดตาม
